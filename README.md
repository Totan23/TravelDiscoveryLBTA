# TravelDiscoveryLBTA

Una aplicación iOS desarrollada en SwiftUI para descubrir destinos turísticos, restaurantes populares y creadores de contenido de viajes. La app presenta una interfaz moderna con secciones para explorar categorías de viajes, destinos populares con imágenes atractivas, recomendaciones de restaurantes con vista de carrusel, y perfiles de usuarios creadores de contenido, todo organizado en una experiencia visual fluida y navegable.
